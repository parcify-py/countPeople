# 📊 Система мониторинга очереди

Веб-приложение для отслеживания количества людей в очереди с использованием YOLOv8 и видеопотокового мониторинга.

## 🎯 Функции

- **Видеопоток в реальном времени** - Отображение видео с камеры со свежим кадром за кадром
- **Интеллектуальное обнаружение** - Детекция людей раз в 5 секунд для оптимизации производительности
- **Время ожидания** - Расчет примерного времени ожидания (30 секунд на человека)
- **Статистика** - Отслеживание макс, мин, среднего количества людей
- **Красивый интерфейс** - Современный дизайн с поддержкой мобильных устройств

## 📋 Требования

- Python 3.8+
- yolov8n.pt (модель YOLOv8)
- Доступ к видеопотоку (HTTP URL)

## 🎨 Интерфейс

- **Сверху**: Примерное время ожидания в очереди
- **Центр**: Видеопоток с обнаруженными людьми (зелёные боксы)
- **Справа**: Текущая статистика
- **Снизу**: Дополнительная информация и статус

## 📊 Отображаемая информация

### Основная статистика
- 👥 **Люди в очереди** - Текущее количество обнаруженных людей
- 🎬 **FPS** - Частота кадров 
- 👥 **Макс. людей** - Максимальное прошлое количество
- 📊 **Средн. людей** - Средний показатель
- 🔍 **Детекций** - Количество выполненных сканирований

## 🔧 Оптимизация

### Почему детекция раз в 5 секунд?
- Экономия ресурсов CPU/GPU
- Плавный видеопоток без замораживаний
- Достаточная частота обновления информации

### Видео
- Кадры передаются каждый раз (не пропускаются)
- Минимальная задержка задана на 5мс
- MJPEG формат для универсальной совместимости

## 💡 Советы

1. Убедитесь, что камера доступна по указанному URL
2. Проверьте интернет соединение при использовании удалённой камеры
3. Для лучшей работы используйте браузер Chrome/Edge/Firefox

## 🐛 Решение проблем

**Видео не загружается:**
- Проверьте URL камеры в `app.py`
- Убедитесь, что камера включена и доступна

**Высокая нагрузка на CPU:**
- Увеличьте `DETECTION_INTERVAL`
- Уменьшите разрешение видео

**Модель не загружается:**
- Скачайте `yolov8n.pt` из Ultralytics
- Поместите в корень папки `countPeople`
